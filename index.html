Demo Interfaces

This is the the demo.dspace.org test site. It hosts demo installations of all DSpace web applications:

  • JSPUI (traditional JSP-based interface)
  • XMLUI (Manakin, the XML / XSLT / Cocoon user interface)
  • OAI-PMH Demo
    • More info on OAI-PMH protocol and its usage.
  • SWORD Service Document (requires login)
    • More info on SWORD protocol and its usage.
  • LNI - Lightweight Network Interface (requires login)
    • More info on the DSpace Lightweight Network Interface (LNI) and its usage.

Give Us Feedback!

Each page of this DSpace Demo site has a red feedback button on the left hand side of the page. You can report issues, or suggest fixes just by clicking on that feedback button from whatever page you are on. You can also see a list of all suggestions on the right hand side of this homepage, and vote on ones you agree with.

Where to Ask Questions

Developers and power-users are available online in IRC (freenode.net #dspace). Feel free to join us by choosing your own "Nickname" below, and clicking "Connect".
