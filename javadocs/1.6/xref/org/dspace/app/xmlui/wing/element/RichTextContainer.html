43  /**
44   * A class representing a character container, such as "p", "hi", "item", or
45   * "cell"
46   * 
47   * This class may not be instantiated on it's own instead you must use one of
48   * the extending classes listed above. This abstract class implements the
49   * methods common to each of those elements.

51   * @author Scott Phillips

62       * Construct a new rich text container.
63       * 
64       * This method doesn't do anything but because the inheriting abstract class
65       * mandates a constructor for this class to compile it must ensure that the
66       * parent constructor is called. Just as implementors of this class must
67       * ensure that this constructor is called, thus is the chain of life. :)
68       * 
69       * @param context
70       *            (Required) The context this element is contained in.
71       */
72      protected RichTextContainer(WingContext context) throws WingException
73      {
74          super(context);
75      }
